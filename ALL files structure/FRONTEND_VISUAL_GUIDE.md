## Animation Timeline

### Page Load (First 1.2 seconds)
```
0.0s - Animated orbs start floating (continuous)
0.0s - Overlay fade in
0.0s - Badge slides down
0.2s - Title fades in with up motion
0.4s - CTA button fades in with up motion
0.6s - Scroll indicator fades in
```

### Feature Cards (Staggered 0-0.6s)
```
0.0s - Card 1 slides up
0.1s - Card 2 slides up
0.2s - Card 3 slides up
0.3s - Card 4 slides up
0.4s - Card 5 slides up
0.5s - Card 6 slides up
```

### Interactive States
```
Hover on Card:
  - Icon floats up and scales +10%
  - Border glows (green highlight)
  - Bottom bar grows from 0 to 60px
  - Box-shadow expands

Hover on Button:
  - Translates -3px (lifts up)
  - Glow expands
  - Background gradient slides in
```
